\begin{abstract}
	In this document we present the results of experiments on three datasets using different classification techniques. Five different classification algorithms were tuned using test datasets and then compared on validation datasets. First we give a brief overview over our experiment environment, then we introduce the datasets and interpret the results. At last we compare the results of all datasets and try to draw a conclusion.
	
\end{abstract}
